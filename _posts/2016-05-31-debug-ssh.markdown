## Enable DEBUG3 to SSH

Put this to your `~/.ssh/config` to set DEBUG3 (`-vvv`) to all SSH connections

```
Host *
  LogLevel DEBUG3
```

Of course this can be given to any host to debug particular host instead of every connection.
